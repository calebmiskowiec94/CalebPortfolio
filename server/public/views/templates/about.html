VeganHelper

Organizes and neatly keeps track of daily meals on a user friendly weekly meal planner

The app has each day listed out which can expand individually into options for Breakfast Lunch and Dinner

Enter foods into the list with the input box

Click the edit button to edit foods.

I am super excited to expand VeganHelper to offer more features for the user, making it an app that is truly suitable for helping someone transition to or maintain a vegan diet.

Technologies: Angular Javascript Node MongoDb Mongoose Express Passport

Aeon Home Survey

Aeon is a housing development company aimed at giving low income families affordable housing in good living conditions

Aeon had a really complicated, uneasy paper survey, that was available in 4 languages for all the thousands of residents

Our solution was to create an electronic version of the survey with features allowing the user to manage the data

My role was working on a lot of the core functionality and setting up the survey views.

Technologies: Angular Javascript Node PostgreSQL Passport Express
